# Labyrint-Oplosser
Jarno, Geal &amp; Ward zoeken uit hoe ze een Labyrint-oplossende AI maken. :P

### Vereiste pakketten

Om het programma goed te laten werken, moet je drie Python-bibliotheken installeren.
Voer elk van onderstaande commando’s uit in je terminal.

---

**NumPy**
Zorgt voor snelle berekeningen met matrices en getallen — essentieel voor het labyrint-algoritme.

```bash
pip install numpy
```

---

**Pandas**
Handig om gegevens te lezen, verwerken of opslaan (bijv. de resultaten van je oplossing).

```bash
pip install pandas
```

---

**Pillow (PIL)**
Wordt gebruikt om afbeeldingen te tekenen of het labyrint visueel weer te geven.

```bash
pip install pillow
```
